**Constants:**
```python
EVAL_SPLIT_DATE = datetime.date(2010, 1, 1)
PHASE2_SPLIT_DATE = datetime.date(2000, 1, 1)
TRAIN_PARQUET = "data/processed/feature_matrix_train.parquet"
TEST_PARQUET = "data/processed/feature_matrix_test.parquet"
FEATURE_COLS_JSON = "data/processed/feature_columns.json"
EVAL_REPORT_PATH = "data/models/eval_report.json"
```

**Logging:** Use `logging.basicConfig(level=logging.INFO, format="%(asctime)s %(levelname)s %(name)s — %(message)s")` and `logger = logging.getLogger("pipeline.model.train_eval")`.

**Step 1: Load and concatenate parquets.**
- Read both parquets with `pd.read_parquet()`
- Concatenate with `pd.concat([train_df, test_df], ignore_index=True)`
- Log row counts for each source and combined total

**Step 2: Internal re-split at 2010-01-01.**
- Use `.apply(lambda d: d < EVAL_SPLIT_DATE)` for date comparison (datetime.date objects, NOT Timestamps)
- `sel_train` = rows with date < 2010-01-01
- `holdout` = rows with date >= 2010-01-01
- Log: `f"Model selection train: {len(sel_train)} rows, Holdout: {len(holdout)} rows"`

**Step 3: Downsample the 2000-2010 slice of sel_train.**
- The pre-2000 slice (from feature_matrix_train.parquet) is ALREADY downsampled 10:1. Do NOT downsample it again.
- The 2000-2010 slice (from feature_matrix_test.parquet) is NOT downsampled. Apply `downsample_negatives()` with ratio=10, random_state=42 to ONLY this slice.
- Import: `from pipeline.features.engineering import downsample_negatives`
- Split sel_train at PHASE2_SPLIT_DATE:
  ```python
  pre2000 = sel_train[sel_train["date"].apply(lambda d: d < PHASE2_SPLIT_DATE)]
  post2000 = sel_train[sel_train["date"].apply(lambda d: d >= PHASE2_SPLIT_DATE)]
  post2000_ds = downsample_negatives(post2000, ratio=10, random_state=42)
  sel_train_final = pd.concat([pre2000, post2000_ds], ignore_index=True)
  ```
- Log: `f"After downsampling 2000-2010 slice: {len(sel_train_final)} training rows"`

**Step 4: Feature column selection.**
- Load `feature_columns.json` into `feature_cols` list (813 columns)
- `X_train = sel_train_final[feature_cols].to_numpy(dtype="float32")`
- `y_train = sel_train_final["EQIndicator"].to_numpy()`
- `X_holdout = holdout[feature_cols].to_numpy(dtype="float32")`
- `y_holdout = holdout["EQIndicator"].to_numpy()`
- Assert: `assert X_train.shape[1] == 813, f"Expected 813 features, got {X_train.shape[1]}"`
- Log shape of X_train, X_holdout

**Step 5: Train both models.**
- LogisticRegression: `LogisticRegression(C=1, penalty="l1", solver="liblinear", max_iter=1000, random_state=42)`
- XGBClassifier: `XGBClassifier(n_estimators=100, max_depth=6, random_state=42, eval_metric="logloss")` — do NOT pass `use_label_encoder` (removed in xgboost 3.x)
- Fit both on X_train, y_train
- Log training completion for each

**Step 6: Evaluate both on holdout.**
- For each model, compute `predict_proba(X_holdout)[:, 1]` to get probabilities
- Compute precision_recall_curve, then F1 array: `2 * precision[:-1] * recall[:-1] / (precision[:-1] + recall[:-1] + 1e-8)`
- `best_idx = np.argmax(f1_scores)`, `threshold = float(thresholds[best_idx])`
- `y_pred = (y_prob >= threshold).astype(int)`
- Compute `f1_score(y_holdout, y_pred)` and `matthews_corrcoef(y_holdout, y_pred)`
- Compute `confusion_matrix(y_holdout, y_pred)` and extract `tn, fp, fn, tp = cm.ravel()`
- Store results for both models in a list

**Step 7: Select winner by MCC.**
- Compare MCC values; winner is the model with highest MCC on holdout
- Log: `f"Winner: {winner_name} (MCC={winner_mcc:.4f}, F1={winner_f1:.4f})"`

**Step 8: Write eval_report.json.**
- Use the winner's threshold (from its own PR curve), not the loser's
- Schema exactly:
```json
{
  "model_used": "LogisticRegression",
  "f1_score": 0.312,
  "mcc": 0.287,
  "threshold": 0.42,
  "eval_split_date": "2010-01-01",
  "confusion_matrix": {"tp": 1240, "fp": 890, "fn": 680, "tn": 45320},
  "both_models": [
    {"model": "LogisticRegression", "f1": 0.312, "mcc": 0.287},
    {"model": "XGBClassifier", "f1": 0.298, "mcc": 0.271}
  ]
}
```
- Write with `json.dump(report, f, indent=2)`
- Log: `f"Evaluation report written to {EVAL_REPORT_PATH}"`

**Step 9: `main()` function wrapping all steps.**
- `os.makedirs("data/models", exist_ok=True)` at the top
- Call all steps sequentially
- Return the winner model object and feature_cols for potential reuse (but primarily this script writes eval_report.json)

3. Deleting the combined DataFrame after splitting into sel_train and holdout
